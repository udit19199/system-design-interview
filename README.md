# system-design-interview

Mock system design interview skill for practice and feedback.

## What it does

- Runs a realistic system design interview
- Asks clarifying and probing questions
- Adjusts difficulty from Junior to Staff
- Ends with a structured debrief
- Can optionally generate a single-file HTML interview simulator UI

## Modes

- `Interview Mode` (default): live interviewer in chat
- `UI Mode` (on request): generates `system_design_interview_simulator.html`

## Use when

- The user wants to practice a system design interview
- The user asks to be interviewed on a system design topic
- The user wants feedback on an architecture answer
- The user wants a mock interviewer for interview prep

## Files

- `SKILL.md` - the main skill instructions
- `references/design-system.md` - visual tokens and layout guidance for UI mode
- `references/interactive-elements.md` - component and interaction patterns for UI mode
